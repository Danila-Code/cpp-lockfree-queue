## Описание
Шаблонный контейнер-очередь без блокировок (lockfree) на основе std::atomic. Критические области не используются. Структура контейнера аналогична односвязному списку и позволяет добавлять элементы в конец и доставать из начала в многопоточном режиме.

## Технологии
* C++ 20
* STL
* Catch2
* CMake

## Особенности
* очередь потокобезопасна
* добавление/удаление элементов в очередь/из очереди происходит без использования мьютексов
* для тестирования методов очереди используется билиотека Catch2
* сложность вставки нового элемента в очередь O(1)
* сложность удаления элемента из очереди O(1)

## Сборка (Linux)
1. Скачайте файлы из текущего репозитория.
2. Создайте в корневой папке проекта папку "build" для сборки проекта, перейдите в неё в командной строке и запустите cmake:\
	`cmake -S . -B build`
3. Запустите сборку проекта в командной строке:\
	`cmake --build build`\
*Проект собран.*

## Запуск (Linux)
* Чтобы запустить программу нужно в командной строке, находясь в папке `build` проекта, ввести:\
	`./LockFreeQueue`

## Использование
Очередь без блокировок - это header only библиотека, которую можно свободно использовать в своих проектах, скопировав файл lock-free-queue.h 

## Системные требования
Компилятор С++, С++20, Catch2Test 2.13.8 (для запуска тестов), CMake 3.8
